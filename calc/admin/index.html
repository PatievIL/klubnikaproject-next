Редактор цен калькулятора открыт только после входа в админ-кабинет

Здесь можно менять цены, константы и значения по умолчанию, а справа сразу видеть, как это влияет на смету, расходы и базовую экономику по модели фермы.

Эта страница не пишет файл на сервер автоматически. Она готовит новый `pricing.json`, который можно скачать, вставить в проект и затем уже зафиксировать в git или CMS.

Без подтверждённой admin-сессии редактор цен, константы и экспортный JSON не показываются.

Сначала войдите в основной админ-кабинет или подтвердите сессию здесь. После этого откроется рабочее пространство калькулятора.

Как работать
  • Меняете значения слева.
  • Смотрите превью справа.
  • Скачиваете новый `pricing.json`.
  • Подменяете файл в проекте или сохраняете через будущую админку.
