# Logo Explorations

В этой папке лежат дополнительные варианты логотипа для `KlubnikaProject`.

Важно:

- это не новые production-логотипы;
- основной для сайта зафиксирован в `docs/brand-ui-visual-guide.md` как текущий digital lockup;
- эти файлы нужны как запасные направления, если позже появится желание упростить или усилить текущее решение.

## Варианты

- `logo-variant-01-horizontal.svg`
  Горизонтальный lockup с ягодой слева и двухстрочным блоком справа.

- `logo-variant-02-inline.svg`
  Более компактная inline-версия для широких header/hero зон.

- `logo-variant-03-badge.svg`
  Badge-подход для аватаров, cover-карточек и брендированных промо-блоков.

Все варианты собраны в фирменной системе:

- `Exo 2`
- `#004d2b`
- `#f5ecdf`
- знак ягоды как основа
